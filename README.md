# Simon Hives Portfolio Canvas

Interaktivni portfolio pro IT support, application support, debugging a technickou komunikaci.

Repository: https://github.com/xhives00/cv

## Spusteni lokalne

Projekt je staticky a nema build krok.

```powershell
python -m http.server 4173
```

Pak otevri `http://localhost:4173`.

## Deploy

Moznosti:

- GitHub Pages: nastavit root repozitare jako static site.
- Render: Static Site, publish directory `.`.
- Railway: nasadit pres jednoduchy static server nebo Nginx.
